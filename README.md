4. 沉浸式体验路线

① 云吸猫入口
打开 [CatPark](https://CatPark.top) → 明暗模式自动跟随系统 → 4K 轮播大图随机撸猫 → 底部运营天数实时滚动。

② 随手做公益
1. 侧边栏 < 5 px 手势唤醒  
2. 点 “志愿者报名” → 选择「投喂组 / 医疗组 / TNR 组」→ 提交后 3 s 弹出证书预览。  
3. 点 “爱心馈赠” → 输入任意金额 → 支付宝 / 微信 / QQ 钱包 / 云闪付四选一 → 留言弹幕实时渲染。

③ 硬核救助指南
Step 1 观察评估 → Step 5 后续安排，每张卡片附带  
- 手绘插图（占位）  
- 可下载 PDF（救助记录表、费用统计、领养协议模板）  
- 紧急联系热区（一键拨号 / 邮件 / 地图导航）

5. 技术实现揭秘

5.1 设计系统
- 主色：`#7A6346`（温暖榛果棕）  
- 辅色：`#E08A34`（活力橙）  
- 圆角统一：`12 px`  
- 阴影层级：`0 4 px 24 px rgba(122,99,70,.08)`  
- 字体：自托管 `Qttf`，中文视觉基线对齐。

5.2 前端架构

文件	职责	
`index.html`	单页路由、SEO 元标签、PWA 清单注入点	
`style.css`	4000+ 行原子级 CSS 变量，支持主题热切换	
`script.js`	零框架、模块化 IIFE、Supabase 实时数据	
`sw.js`（预留）	离线缓存、推送订阅、后台同步	

5.3 数据与部署
- Database：Supabase Postgres + Row Level Security  
- Storage：猫咪图片 CDN 加速，自动 WebP 转化  
- CI/CD：GitHub Actions → Cloudflare Pages，分支即环境  
- 监控：Web-Vitals + Sentry + Cloudflare Analytics
